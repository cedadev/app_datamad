File names must follow the format:

<GRANT REFERENCE (underscore separated)>_<DOC TYPE>
e.g. NE_T000619_1_DMP.pdf (NERC grant ref.) or 12345_DMP.pdf (UKRI grant ref)

Document types used are:

  • ODMP - Outline data management plan
  • DMP - Data management plan
  • CFS - Case for support
  • TA - Technical appraisal
  • JOR - Justification of resources
  • PROFORMA - Grant proposal
